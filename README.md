# Jib's Recruiter Portfolio

A modern, minimalist, and developer-centric portfolio website designed for a recruiter.

## Features
- **Dark/Light Mode:** Toggle between themes using the sun/moon icon.
- **Smooth Animations:** Sections fade in on scroll using the Intersection Observer API.
- **Responsive Design:** Optimized layout for desktop and mobile devices.
- **Modern Aesthetic:** Built with JetBrains Mono and a clean, high-contrast color palette.

## Tech Stack
- HTML5
- CSS3 (Custom properties, CSS Grid)
- Vanilla JavaScript
